# CSS Properties Used in Buttons

## 1. Display Flexbox
Flexbox is a layout model that makes it easier to design a complex layout structure without using float or positioning. By using `display: flex;`, you can align buttons and other elements horizontally or vertically.

## 2. Padding, Margin, and Border (Box Model)
- **Padding**: This is the space between the content and the border of the button. Example: `padding: 10px 15px;`
- **Margin**: This is the space outside the button. Example: `margin: 5px;`
- **Border**: Defines the boundary of the button. Example: `border: 1px solid #ccc;`

## 3. Colors and Gradients
CSS allows for both solid colors and gradients. 
Example of a solid color button: `background-color: #4CAF50;`
Example of a gradient background: `background: linear-gradient(to right, #4CAF50, #45a049);`

## 4. Cursor Pointer
Using `cursor: pointer;` changes the cursor to a hand when hovering over the button to indicate it's clickable.

## 5. Transition and Transform
- **Transition**: Allows you to change property values smoothly (over a given duration).
Example: `transition: background-color 0.3s ease;`
- **Transform**: Used to rotate, scale, skew, or translate an element.
Example: `transform: scale(1.1);`

## 6. Box Shadow
Adds shadow effects around an element. Example: `box-shadow: 0 4px 8px rgba(0,0,0,0.2);`

## 7. Hover Pseudo-Class
The `:hover` pseudo-class applies styles when the user hovers over the button. Example: `button:hover { background-color: #3e8e41; }`

## 8. Active Pseudo-Class
The `:active` pseudo-class applies styles when the button is being clicked. Example: `button:active { transform: scale(0.95); }`

## 9. Disabled State
Styles for a disabled button using the `:disabled` pseudo-class to indicate it cannot be interacted with. Example: `button:disabled { opacity: 0.5; cursor: not-allowed; }`

## 10. Responsive Media Queries
Media queries allow you to apply different styles for different devices. Example: 
```css
@media (max-width: 600px) {
  button { font-size: 14px; }
}
```
